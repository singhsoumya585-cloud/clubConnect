# Club Connect: Student Organization Hub

## Overview
**Club Connect** is a mobile application built with **Flutter** for the BPA V05 Mobile Applications competition.  
It helps students manage and participate in school clubs by allowing them to:

- Browse and join clubs
- View club events and announcements
- Track participation
- Communicate with club leaders
- Admins can create and manage clubs and events

The app uses Firebase Firestore to store club and event data and includes simulated calendar and messaging APIs for demonstration purposes.

## Features

- **Firebase Integration** – Clubs, events, and messaging stored in Firestore.  
- **Admin Panel** – Create and edit clubs and events.  
- **Calendar API** – View upcoming events per club.  
- **Messaging API** – Communicate with club members.
